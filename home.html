    Gitte Jan 2021 Github
    I learned making CLI application using OCLIF. The idea was to automate my regular task using code and command line. I wanted to automate that part of my regular day where I open my command line and I need to put a lot of efforts in just changing my path to the actual directory read more C/user/dell/documents/web dev/projects/personal projects/automation/cliapp/gitty. This is insane. I end up wasting my 3-5min in just changing the path.(tho we have arrow up, I don't use that much.) So the idea was when I open my command line, with just one short command I should be redirected to my working directory. I spend 3 days doing research, that was kind of the best part. During this research I got to know about Child Process in node js (It helps in executing command line commands through node application.) This was what I was finding BUT I was not able to execute cd command. (maybe coz cd is not an executable command) Other commands worked. I then thought of making a git command automation, because we programmers end up spending a lot of time in pushing the code to github. Gitte - With just one command you can init git, add all your files to stagging area, commit it, add remote, push it. $git push -r add remote -m commit-msg This was really usefull. I think you should give a try, Just you need to install it globally i.e npm install -g gitte (Read the readme.md file for usage) read less
